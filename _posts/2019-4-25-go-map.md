1. 无序键值对集合，字典要求key必须是支持相等运算符（==，!=)的数据;
2. 引用类型,使用make函数或初始化表达语句来创建

Demo:


```
func main(){
	m := make(map[string]int)
	m["a"] = 1
	m["b"] = 2

	m2 := map[int]struct{
		x int
	}{
		1: {x:100},
		2: {x:200},
	}
	fmt.Println(m, m2)

	m["a"] = 3
	m["c"] = 4 

	if v, ok := m["d"];ok{
		println(v)
	}
	delete(m, "d")
}

```
